# AI Impact on Jobs 2030

## 📌 Project Overview
This PMA project is use to analyze the impact of Artificial Intelligence on jobs by 2030 using machine learning models.  
The dataset (`AI_Impact_on_Jobs_2030.csv`) includes job titles, industries, wages, automation risk scores, and growth rates.  
The goal is to predict which jobs are most at risk of automation and understand future employment trends.
